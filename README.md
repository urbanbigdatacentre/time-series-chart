## Run Locally

Clone the project

```bash
  git clone https://github.com/urbanbigdatacentre/time-series-chart.git
```

Go to the project directory

```bash
  cd time-series-chart
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
